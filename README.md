Recipe Finder

A responsive web application that allows users to search for recipes and view detailed cooking instructions, ingredients, category information, and meal origin using data fetched from TheMealDB API.

Live demo
https://alishbanazim.github.io/recipe-finder/

Overview

Recipe Finder helps users discover meals by simply entering a recipe name. The application retrieves real-time data from an external API and presents it in a clean and user-friendly interface.



Features

Search meals by name

Real-time API integration

Display meal images

Show meal category and country

View complete ingredients list

Display cooking instructions

Responsive design for mobile and desktop devices

User-friendly interface

Technologies Used

HTML5
